Політика конфіденційності

Дата набрання чинності: 03 червня 2026 р.

1. Які дані ми збираємо

Ми збираємо такі персональні дані, які ти надаєш добровільно при заповненні квізу:

  • Ім'я (необов'язково)
  • Email-адреса
  • Дата, час та місце народження
  • Відповіді на питання квізу

Також ми автоматично збираємо технічні дані: IP-адреса, тип браузера, UTM-параметри для аналітики.

2. Як ми використовуємо дані

  • Для генерації персональної розшифровки Дизайну Людини
  • Для надсилання розшифровки на твою email-адресу
  • Для надсилання пов'язаних пропозицій (upsell) — ти можеш відписатися
  • Для аналізу та покращення нашого сервісу

3. Передача даних третім особам

Ми передаємо твої дані лише наступним сервісам для виконання замовлення:

  • LiqPay — обробка платежів (не зберігають дані картки у нас)
  • Anthropic Claude API — генерація тексту розшифровки (дата/місце народження)
  • Brevo — відправка email-листів
  • Supabase — зберігання даних замовлення

Ми не продаємо та не передаємо твої дані іншим третім особам.

4. Зберігання даних

Твої дані зберігаються в зашифрованому вигляді на серверах Supabase (EU-регіон). Ми зберігаємо їх протягом 2 років або до твого запиту на видалення.

5. Твої права

  • Право на доступ до своїх даних
  • Право на виправлення даних
  • Право на видалення даних ("право на забуття")
  • Право на відмову від маркетингових розсилок

Для реалізації прав — напиши нам на email нижче.

6. Cookie та аналітика

Ми використовуємо Google Analytics 4 та Meta Pixel для аналізу відвідуваності та ефективності реклами. Ці інструменти збирають анонімізовані дані про поведінку на сайті.
