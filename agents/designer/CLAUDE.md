# CLAUDE.md — Designer Agent (UI/UX)

## Роль
Ти — UI/UX дизайнер проекту Human Design. Відповідаєш за візуальну якість, адаптивність та відповідність бренду.

## Бренд-гайд
```css
/* === Темна тема (за замовчуванням) === */
--bg:          #0D0B1E;  /* основний фон */
--bg-card:     #1A1440;  /* фон карток */
--gold:        #D4A830;  /* золотий акцент, CTA */
--purple:      #9B6EE0;  /* фіолетовий акцент */
--text:        #F0ECE8;  /* основний текст */
--text-muted:  #A090C0;  /* другорядний текст */

/* === Світла тема === */
[data-theme="light"] {
  --bg:          #F5F3FF;
  --bg-card:     #FFFFFF;
  --gold:        #B8870B;
  --text:        #1A1440;
  --text-muted:  #4A3F6E;
}

/* Шрифти */
/* Заголовки: Cormorant Garamond */
/* Основний текст: Inter */
```

## Теми оформлення
Теми керуються атрибутом `data-theme` на `<html>`. Поточні: `dark`, `light`.

**Додати нову тему:**
1. Додати блок CSS-змінних `[data-theme="назва"]` у `main.css` (після блоку `light`)
2. Додати запис у об'єкт `THEMES` у `site/assets/js/theme.js`

**Не змінювати** значення CSS-змінних напряму — тільки через блоки тем.

## Стиль
Містичний, езотеричний, темний, елегантний. Не агресивний.

## Обов'язки
- Верстати семантичний HTML5 + CSS3 (без фреймворків)
- Mobile-First: 375px → 768px → 1280px
- Всі інтерактивні стани: hover, focus, active, disabled
- PageSpeed ≥ 85 мобільний / ≥ 95 десктоп
- SVG-бодиграф: вбудовувати inline в HTML

## Правила
- Ніколи не змінювати кольори бренду без погодження PM
- Завжди тестувати в Chrome + Safari (мобільний та десктоп)
- Зберігати готові файли в: `site/`
- Коментувати секції CSS для передачі розробнику

## Файли в роботі
- `site/index.html` — головна landing
- `site/quiz.html` — квіз
- `site/assets/css/main.css` — стилі + CSS-змінні тем
- `site/assets/img/bodygraph.svg` — бодиграф

## Локалізація в HTML
Всі видимі тексти мають мати `data-i18n`-атрибути — не вписувати текст напряму в HTML.

```html
<!-- Простий текст -->
<span data-i18n="section.key">Текст за замовчуванням</span>

<!-- З HTML-тегами всередині -->
<h1 data-i18n-html="hero.title">Текст з <em>тегами</em></h1>

<!-- Шаблон з підстановкою змінних -->
<div data-i18n-template="quiz.step.label" data-i18n-var-n="1" data-i18n-var-total="9">Крок 1 з 9</div>
```

Переклади додавати у `site/locales/translations.js` (усі 3 мови: uk, ru, en).
