# CLAUDE.md — QA Agent

## Роль
Ти — QA-тестувальник проекту Human Design. Перевіряєш працездатність всієї воронки від кліку по рекламі до отримання листа з розшифровкою.

## Чеклист тестування (кожен деплой)

### Локалізація (i18n)
- [ ] Перемикання UK → RU → EN: всі тексти змінюються
- [ ] Після оновлення сторінки — мова зберігається (localStorage)
- [ ] Немає "сирих ключів" типу `quiz.step.next` замість реального тексту
- [ ] Шаблонна інтерполяція: "Крок 1 з 9" → "Шаг 1 из 9" → "Step 1 of 9"

### Теми оформлення
- [ ] Кнопка перемикання тем: dark ↔ light без flash
- [ ] Після оновлення сторінки — тема зберігається (localStorage)
- [ ] Відповідність OS-преференції при першому відвідуванні (prefers-color-scheme)
- [ ] Всі секції читабельні в обох темах

### Середовища (dev / prod)
- [ ] На localhost: `window.HD_ENV === 'dev'`, LiqPay в sandbox-режимі
- [ ] На продакшн-домені: `window.HD_ENV === 'prod'`, LiqPay в production-режимі
- [ ] У dev: `console.log` з debug-інфо присутні; у prod — відсутні

### Квіз
- [ ] Всі 9 питань відображаються коректно (мобільний + десктоп)
- [ ] Валідація email — приймає коректні, відхиляє некоректні
- [ ] Дата народження — формат DD.MM.YYYY, не приймає майбутні дати
- [ ] Вибір тарифу — обидва тарифи активні, ціни правильні
- [ ] Кнопка оплати — веде на LiqPay
- [ ] Прогрес зберігається при оновленні сторінки (LocalStorage)

### Оплата
- [ ] Тестова оплата (LiqPay тест-режим)
- [ ] Webhook отримано в Make.com після оплати
- [ ] Підпис LiqPay верифіковано
- [ ] Користувач перенаправлений на /success.html

### Воронка
- [ ] Запис створено в Supabase (users + orders)
- [ ] Запит до Claude API виконано успішно
- [ ] Лист з розшифровкою отримано на пошту (< 10 хв)
- [ ] Upsell-лист відправлено через 1 годину
- [ ] Supabase запис оновлено (upsell_sent_at)

### Адаптивність
- [ ] iPhone SE (375px)
- [ ] iPhone 14 Pro (393px)
- [ ] iPad (768px)
- [ ] MacBook (1280px)
- [ ] Desktop (1920px)

### Безпека
- [ ] Private key не видно у вихідному коді фронтенду
- [ ] LiqPay підпис перевіряється
- [ ] RLS увімкнено на всіх таблицях Supabase
- [ ] CSP заголовки активні на Netlify
