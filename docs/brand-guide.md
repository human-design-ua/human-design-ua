# Human Design UA — Brand Guide

## Назва та позиціонування

**Назва:** Дизайн Людини / Human Design UA  
**Слоган:** "Ти не зламана — ти просто не знала свій дизайн"  
**Позиція:** Персоналізовані розшифровки Дизайну Людини для жінок, які шукають себе

## Цільова аудиторія

**Основна:** Жінки 25–45 років, Україна  
**Інтереси:** психологія, езотерика, саморозвиток, астрологія  
**Болі:** хронічна втома, складно приймати рішення, "живу не своїм життям"  
**Рівень знань HD:** від "вперше чую" до "практикую"

## Типографіка

### Шрифти
- **Заголовки (H1–H3):** Cormorant Garamond — елегантний, серифний, містичний
- **Основний текст:** Inter — сучасний, читабельний, нейтральний

### Ієрархія
```css
H1: clamp(2.2rem, 6vw, 3.8rem) — Cormorant Garamond 400/700
H2: clamp(1.8rem, 4vw, 3rem)   — Cormorant Garamond 400
H3: clamp(1.1rem, 2.5vw, 1.5rem) — Cormorant Garamond 400
Body: 1rem / 1.65 — Inter 400
Small: 0.85rem — Inter 400/500
```

### Курсив
Використовується для емоційних акцентів у заголовках: `<em>` тег → колір золотий (#D4A830)

## Голос бренду (Tone of Voice)

### Характер
- Тепла подруга, що добре знається на HD
- Мудра, але не снобська
- Конкретна, без "духовного туману"
- Підтримуюча, не маніпулятивна

### Що говоримо
✓ "Дізнайся, чому ти приймаєш рішення саме так"  
✓ "Розшифровка покаже джерело твоїх паттернів"  
✓ "Починаєш застосовувати вже завтра"  

### Що НЕ говоримо
✗ "Тільки сьогодні! Дефіцит!"  
✗ "Ти точно знайдеш себе і будеш щасливою"  
✗ "Інакше так і будеш страждати"  
✗ Медичні або психологічні діагнози  

## Візуальний стиль

**Загальний тон:** Темний, містичний, елегантний  
**Відчуття:** Як нічне небо + золоте світло. Глибина, але не страшно.  
**Референси:** Мінімалістична езотерика, dark academia, luxury wellness

### Іконографія
- Символи: ✦ (головний символ бренду), ◈, ☽, ✨
- Геометрія: шестикутники, трикутники, ромби
- Не використовувати: emoji у заголовках, яскраві кольори, stock photos

### Зображення
- Формат: WebP для всіх зображень
- Стиль: темні, атмосферні, мінімалістичні
- Бодиграф: inline SVG, золоті та фіолетові кольори на темному фоні

## Компоненти

### Кнопки
- Primary (gold): `#D4A830` background, dark text — для головних CTA
- Secondary (outline): золота рамка, прозорий фон — для вторинних дій
- Min height: 44px (WCAG touch target)

### Картки
- Background: `#1A1440`
- Border: `rgba(212, 168, 48, 0.12)` у спокої
- Border hover: `rgba(212, 168, 48, 0.4)`
- Border radius: 12–20px
- Padding: 1.75–2rem

### Badges
- Border: 1px solid gold або purple
- Border radius: 100px (pill)
- Font size: 0.8rem uppercase
