## 2. Title та Description теги

### Головна сторінка (index.html)

```
Title (50-60 симв.):
Дизайн Людини — Персональна Розшифровка Бодіграфу | від 399 грн

Description (150-160 симв.):
Дізнайся свій тип, авторитет та профіль Дизайну Людини. Персональна
розшифровка бодіграфу за датою народження — готова за 10 хвилин на пошту.
```

### Сторінка квізу (quiz.html)

```
Title:
Розрахувати Дизайн Людини Онлайн — Квіз за 3 хвилини

Description:
Пройди короткий опитник і отримай персональну розшифровку свого Дизайну
Людини. Дата, час та місце народження — і твоя карта готова за 10 хвилин.
```

### Upsell сторінка (upsell.html)

```
Title:
Повна Розшифровка Дизайну Людини — Канали, Хрест, Plan на 90 Днів

Description:
Поглиблений аналіз бодіграфу: всі центри, канали, гексаграми, інкарнаційний
хрест та особистий план на 90 днів. Розшифровка ~50 сторінок на пошту.
```

## 4. Рекомендації по контенту

### Принципи (без переспаму)

- Основний ключ `дизайн людини` — 1 раз у H1, 2-3 рази у тексті на 1000 слів
- LSI-слова вживати природно — `бодіграф`, `авторитет`, `стратегія`, `тип`
- Кожен H2/H3 — окремий пошуковий запит (довгий хвіст)
- Alt-теги зображень: `бодіграф дизайн людини`, `енергетична карта особистості`
- URL структура: `/`, `/quiz`, `/upsell` — стислі, без кирилиці

### Пріоритет сторінок для просування

| Сторінка | Ключовий запит | Мета |
|---|---|---|
| `/` (головна) | дизайн людини / розшифровка бодіграфу | Трафік + конверсія |
| `/quiz` | розрахувати дизайн людини онлайн | Транзакція |
| `/upsell` | повна розшифровка дизайну людини | Апселл |
